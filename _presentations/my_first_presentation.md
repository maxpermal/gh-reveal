## Intro
On utilise le markdown pour la mise en forme.
On débute une nouvelle diapositive avec le mot clé "§ new" (sans espace)

§new

## Faire des notes
Pour faire des notes, c'est "§ notes" (sans espace)

S pour voir les notes

§notes
Ceci est une note
